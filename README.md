# Step1: Creat An Empty Project

First we need to install dva-cli,

    npm install -g dva-cli


Second, create an empty project,

    dva new user-dashboard
    cd user-dashboard
    npm start

Now open <http://localhost:8989> in browser we'll see it's running!
